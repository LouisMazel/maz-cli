> CLI Tool to create Nuxt/Prismic project

## Install

```bash
npm install -g maz-cli
```

or

```bash
yarn global add maz-cli
```

## Create new project

```bash
maz-cli create <app-name>
```

## Features & prerequisites

1. Choose the package manager
    - Yarn
    - Npm

## CLI Options

### `--help`
